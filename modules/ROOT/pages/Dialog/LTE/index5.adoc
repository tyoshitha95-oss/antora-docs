== Getting Started

To begin:

. Configure the RADIUS/AAA server for LTE authentication.
. Register subscriber identifiers (SIM/IMSI or device MAC).
. Define LTE location policies and access parameters.
. Configure data quotas, bandwidth profiles, and FUP thresholds.
. Monitor subscriber sessions, traffic usage, and network logs.
