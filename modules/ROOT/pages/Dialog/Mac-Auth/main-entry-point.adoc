= Main Entry Point

== Main Entry Point: handle_request

=== Purpose
Main dispatcher for RADIUS requests.

=== Parameters
* `$self` – Object instance
* `$p` – RADIUS packet object
* `$dummy` – Placeholder parameter
* `$extra_checks` – Additional validation flags

=== Logic Flow

[source,pseudocode]
----
FUNCTION processRequest(request):

    IF request.code == "Access-Request" THEN
        IF authenticateUser(request) <= 0 THEN
            RETURN REJECT
        END IF
        adjustReply(request)
        RETURN ACCEPT

    ELSE IF request.AcctStatusType == "Start" THEN
        accStart = handleAccountingStart(request)
        IF accStart == 1 THEN
            RETURN ACCEPT
        ELSE
            RETURN REJECT
        END IF

    ELSE IF request.AcctStatusType == "Stop" THEN
        accStop = handleAccountingStop(request)
        IF accStop == 1 THEN
            RETURN ACCEPT
        ELSE
            RETURN REJECT
        END IF

    ELSE IF request.AcctStatusType == "Alive" THEN
        handleAccountingAlive(request)
        RETURN ACCEPT

    ELSE
        RETURN REJECT

END FUNCTION
