= Authentication Handler

== Authentication Handler: authenticateUser

=== Purpose
Core authentication logic for `Access-Request` packets.

=== Key Variables
* `$mac` – MAC address from `User-Name`
* `$serviceInfo` – `Huawei-Service-Info` attribute
* `$sessionid` – `NAS-Identifier:Acct-Session-Id`
* `@locdata` – Location data array
* `@restriction` – Time restriction data

=== Logic Flow

== Logic Flow: User Identification
:toc:
:toclevels: 2

== Step 1: Location and Time Validation

[source,pseudocode]
----
BEGIN

GET locationData using getLocation2(self, p)
GET restrictionData using getTimespan(self, p, locationData[7])

IF locationData[0] == 0 THEN
    LOG "Location Not Found"
    RETURN 0
END IF

IF restrictionData[0] == 0 THEN
    LOG "Time Restricted"
    RETURN 0
END IF

CONTINUE

END
----

== Step 2: Initial Login Detection

[source,pseudocode]
----
BEGIN

IF serviceInfo does NOT start with "N" THEN
    // Initial Login Request
    PROCESS initial login
ELSE
    // Session Continuation
    PROCESS session continuation
END IF

END
----

== Step 3: User Existence Check and Quota Management

[source,pseudocode]
----
BEGIN

IF locationData[6] == 1 THEN
    // Daily Reset Enabled

    SET query =
        SELECT user data,
        (CURRENT_DATE - LAST_RESET_TIME - adjusted daily offset) AS DATEDIFF
        FROM MACL2USERS
        WHERE MAC = mac AND LOCID = locid

ELSE
    // No Daily Reset

    SET query =
        SELECT user data,
        (CURRENT_TIME - LAST_RESET_TIME) AS DATEDIFF
        FROM MACL2USERS
        WHERE MAC = mac AND LOCID = locid

END IF

EXECUTE query

END
----
