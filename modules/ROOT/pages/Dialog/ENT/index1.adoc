== Getting Started

To begin:

. Configure the RADIUS server for enterprise authentication.
. Register enterprise subscriber accounts or device MAC addresses.
. Configure enterprise VLANs and location mappings.
. Define access policies, bandwidth limits, and security rules.
. Monitor enterprise sessions, authentication logs, and usage reports.
