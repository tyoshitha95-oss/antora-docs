= Public Dynamic Profile Test Case
:toc: left
:toclevels: 3
:icons: font
:sectnums:

== Description
Validates the authentication and assignment flow for a standard GPON subscriber dynamically assigned a public IP profile. The request is processed and dynamically provisioned by the backend APIs/Radius engine.

== Configuration Mapping

The following test case mapping demonstrates the specific parameters supplied to the AAA backend API:

[cols="1,2", options="header"]
|===
| Parameter | Value

| Endpoint ID
| `94666543212`

| Authentication Match
| `MySecurePass123`

| Assigned Group
| `GPON`

| customnumfield1
| `150`

| sessionlimit
| `300`

| barringstatus
| `0` (Unbarred - Allowed)

| status
| `1` (Active)

| Framed-Pool
| `PUBLIC_DYNAMIC`
|===

== Test Implementation Request

[source, bash]
----
curl --location --request PUT 'http://localhost:4000/users/94666543213' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--header 'Authorization: ••••••' \
--data-urlencode 'password=MySecurePass123' \
--data-urlencode 'customfield13=framepoolIdvalue' \
--data-urlencode 'customnumfield1=150' \
--data-urlencode 'sessionlimit=300' \
--data-urlencode 'groupname=GPON' \
--data-urlencode 'barringstatus=0' \
--data-urlencode 'status=1'
----
