= Public Dynamic IP Assignment

This use case defines GPON user creation using a dynamic public IP pool.

== Description

The user is assigned an IP dynamically from a predefined pool using a framed pool identifier.

== Additional Parameter

[cols="1,2,2"]
|===
|Parameter |Description |Example

|customfield13
|Framed Pool ID used for dynamic public IP allocation
|PUBLIC_DYNAMIC / Loopback100
|===

== Behavior

* IPv4 address is dynamically assigned from a public pool
* Pool is identified using customfield13
* No static IP is defined
* No IPv6 prefix is included unless separately configured

== Example CURL

[source,bash]
----
curl --request PUT \
--url 'http://10.62.167.228:80/users/94367526732' \
--header 'Content-Type: application/x-www-form-urlencoded' \
--header 'Accept: application/json' \
--header 'Authorization: Basic <token>' \
--data 'customfield11=94367526732&password=EonPr73637&groupname=GPON&customfield10=94367526732&customfield12=DNET31&maxusersessions=1&sessionlimit=172800&customnumfield1=86400&customfield13=Loopback100'
----
